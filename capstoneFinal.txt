Drilling simple git/hub sequence as Capstone:

1. Clone github repo (git clone)
2. Create & switch to ‘feature’ branch (git switch -c)
3. Modify files and stage changes (git add .)
4. Confirm staging (git status)
5. Simulate mistake and resolve (restore --staged)
6. Re-stage and commit changes (git commit -m)
7. View commit history (git log --oneline)
8. Undo bad [unpushed] commit (reset --soft HEAD~1)
9. Re-commit and push changes (git push -u origin)
10. PR -> merge on GitHub -> modify file
11. Switch to main and pull latest changes (git pull)
12. Clean workspace (git branch -d)
